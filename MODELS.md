## 모델 변경 방법

`.env` 파일에서 `OPENAI_MODEL` 값 수정:

```env
# 최적 균형 (추천)
OPENAI_MODEL=gpt-4o-mini

# 최고 성능
OPENAI_MODEL=gpt-4o

# 빠른 속도, 낮은 품질
OPENAI_MODEL=gpt-3.5-turbo
```
