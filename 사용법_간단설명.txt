=====================================
    GPT-4o 채팅 프로그램 사용법
=====================================

【필수 설치】
1. Node.js 설치 (한 번만)
   - https://nodejs.org 접속
   - LTS 버전 다운로드
   - 설치 (Next 계속 클릭)

【실행 방법】
1. "자동실행.bat" 더블클릭
2. 브라우저가 자동으로 열림
3. 비밀번호: admin123 입력
4. 채팅 시작!

【종료 방법】
- 검은 창(명령 프롬프트) 닫기

【문제 해결】
- 안 될 때: Node.js 설치 확인
- 포트 충돌: 다른 프로그램 종료

【파일 설명】
- 자동실행.bat → 메인 실행 파일
- 설치및실행_간편버전.bat → 더 간단한 버전
- START_CHAT.bat → 영문 버전
